# Musicplayer-using-JavsScript

A Javascript project for playing music.
In this project you will learn working of user-defined function and in-built functions of javascript.

# To run this on your pc

Download the given files and folder. And keep them in a single folder. And run music_player.html then you can see the page with music tabs single click to play and double click to pause.

# function

fuctions used are Audio, queryselectorall("classname") : in this it will select all the elements which has same classname
.play / .pause : this will play and pause the song.
